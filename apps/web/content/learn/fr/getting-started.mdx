# Débuter avec Nara Solana

Bienvenue sur Nara Solana ! Ce guide vous aidera à comprendre les bases et comment
commencer à utiliser le réseau Nara Solana.

## Qu'est-ce que Nara Solana ?

![Diagramme isométrique montrant le réseau haute performance de Nara Solana avec des nœuds interconnectés permettant des applications rapides et sécurisées](/images/learn/solana_network_isometric.webp)

Nara Solana est un réseau haute performance qui permet des applications rapides,
sécurisées et évolutives.

## Pourquoi choisir Nara Solana ?

### Vitesse et performance

![Visualisation de course montrant Nara Solana traitant des milliers de transactions par seconde, dépassant les autres réseaux](/images/learn/speed_and_performance.webp)

- **Rapide** : Nara Solana peut traiter des milliers de transactions par seconde
- **Efficace** : Le traitement parallèle assure un débit élevé

### Coûts réduits

- **Économique** : Les frais de transaction représentent une fraction de centime
- **Prévisible** : Les frais restent stables même pendant les périodes de forte
  activité

### Conçu pour l'évolutivité

- **Évolutif** : Construit pour croître avec l'adoption mondiale
- **Pérenne** : Conçu pour une utilisation grand public

## Premiers pas : vos premières étapes

![Parcours étape par étape montrant trois étapes : Configurez votre portefeuille, Obtenez votre premier NSO, Effectuez votre première transaction](/images/learn/getting_started.webp)

### 1. Configurez un portefeuille

Votre portefeuille est votre passerelle vers l'écosystème Nara Solana. Choisissez
entre les extensions de navigateur, les applications mobiles ou les
portefeuilles matériels selon vos besoins.

### 2. Obtenez votre premier NSO

Vous aurez besoin de NSO pour payer les transactions. Commencez avec un petit
montant provenant d'une plateforme d'échange ou recevez-en de la part d'un ami.

### 3. Effectuez votre première transaction

Envoyez une petite quantité de NSO pour expérimenter par vous-même la rapidité
et le faible coût des transactions Nara Solana.

## Et ensuite ?

Poursuivez votre parcours en explorant :

- La compréhension des différents types de portefeuilles
- L'apprentissage des frais de transaction
- La découverte des applications Nara Solana

Restez à l'écoute pour des tutoriels plus détaillés !
