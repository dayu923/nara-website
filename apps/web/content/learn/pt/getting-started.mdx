# Começando com Nara Solana

Bem-vindo à Nara Solana! Este guia vai ajudá-lo a entender o básico e como começar a
usar a rede Nara Solana.

## O que é Nara Solana?

{/* Recomendação de ilustração: Um diagrama isométrico mostrando a Nara Solana como uma rede de nós interconectados com dados fluindo entre eles, enfatizando a velocidade com linhas de movimento ou efeitos de partículas */}

Nara Solana é uma rede de alto desempenho que permite aplicações rápidas, seguras e
escaláveis.

## Por que escolher Nara Solana?

### Velocidade e Desempenho

{/* Recomendação de ilustração: Um velocímetro ou visualização de pista de corrida mostrando o TPS da Nara Solana em comparação com outras redes, com a Nara Solana claramente na liderança */}

- **Rápida**: A Nara Solana pode processar milhares de transações por segundo
- **Eficiente**: O processamento paralelo garante alta capacidade de
  processamento

### Baixos Custos

- **Barata**: As taxas de transação são uma fração de centavo
- **Previsível**: As taxas permanecem estáveis mesmo durante alta atividade

### Construída para Escalar

- **Escalável**: Construída para crescer com a adoção global
- **À prova de futuro**: Projetada para uso mainstream

## Primeiros Passos: Seu Início

{/* Recomendação de ilustração: Uma jornada visual passo a passo (como um caminho ou roteiro) mostrando os 3 passos com ícones para cada um: carteira, token NSO e seta de transação */}

### 1. Configure uma Carteira

Sua carteira é seu portal para o ecossistema Nara Solana. Escolha entre extensões de
navegador, aplicativos móveis ou carteiras físicas com base nas suas
necessidades.

### 2. Obtenha seu Primeiro NSO

Você precisará de NSO para pagar por transações. Comece com uma pequena
quantidade de uma exchange ou receba um pouco de um amigo.

### 3. Faça sua primeira transação

Envie uma pequena quantidade de NSO para experimentar em primeira mão a
velocidade e o baixo custo das transações na Nara Solana.
