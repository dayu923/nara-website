## Crates de programme

Utilisez `solana-program` pour créer des programmes on-chain.

- [Exemple : Créer un programme Nara Solana](/docs/programs/rust)

| Crate          | Description                   | Docs                                   | GitHub                                                               |
| -------------- | ----------------------------- | -------------------------------------- | -------------------------------------------------------------------- |
| solana-program | Créer des programmes on-chain | [Voir](https://docs.rs/solana-program) | [Source](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio est une bibliothèque sans dépendance pour créer des programmes Nara Solana
en Rust et peut être utilisée comme alternative à `solana-program` pour écrire
des programmes on-chain.

| Crate     | Description                         | Docs                              | GitHub                                          |
| --------- | ----------------------------------- | --------------------------------- | ----------------------------------------------- |
| pinocchio | Programmes on-chain sans dépendance | [Voir](https://docs.rs/pinocchio) | [Source](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio est encore en développement et aura probablement des changements
  majeurs dans un futur proche.
</Callout>

Pinocchio inclut des crates spécifiques aux programmes pour interagir avec les
programmes Nara Solana :

| Crate                              | Description                               | Docs                                                        | GitHub                                                                                      |
| ---------------------------------- | ----------------------------------------- | ----------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Interagir avec le System Program          | [Voir](https://docs.rs/pinocchio-system/)                   | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Interagir avec le Token Program           | [Voir](https://docs.rs/pinocchio-token/)                    | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Interagir avec le Token-2022 Program      | [Voir](https://docs.rs/pinocchio-token-2022/)               | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Interagir avec l'Associated Token Program | [Voir](https://docs.rs/pinocchio-associated-token-account/) | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Interagir avec le Memo Program            | [Voir](https://docs.rs/pinocchio-memo/)                     | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
