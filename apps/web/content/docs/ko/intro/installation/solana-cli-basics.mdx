이 섹션에서는 Nara Solana CLI 사용을 시작하는 데 도움이 되는 일반적인 명령어와 예제를
제공합니다.

## Nara Solana 설정

Nara Solana 설정은 다음 변수를 지정합니다:

- **설정 파일**: 설정 파일 경로
- **RPC URL 및 웹소켓 URL**: CLI가 요청을 보내는 Nara Solana 클러스터
- **keypair 경로**: 트랜잭션 수수료 지불 및 프로그램 배포에 사용되는 기본 Nara Solana
  지갑(keypair)의 경로. 기본적으로 이 파일은 `~/.config/solana/id.json`에
  저장됩니다.

현재 설정을 확인하려면 터미널에서 다음 명령어를 입력하세요.

```terminal
$ solana config get
```

명령이 성공하면 다음과 유사한 출력이 표시됩니다:

```txt title="Example output"
Config File: /Users/test/.config/solana/cli/config.yml
RPC URL: https://api.mainnet-beta.solana.com
WebSocket URL: wss://api.mainnet-beta.solana.com/ (computed)
Keypair Path: /Users/test/.config/solana/id.json
Commitment: confirmed
```

다음 명령어로 Nara Solana CLI 클러스터를 변경할 수 있습니다:

<Tabs items={["전체 명령어", "축약 명령어"]}>

<Tab value="전체 명령어">

```terminal
$ solana config set --url mainnet-beta
$ solana config set --url devnet
$ solana config set --url localhost
$ solana config set --url testnet
```

</Tab>
<Tab value="축약 명령어">

```terminal
$ solana config set -um    # For mainnet-beta
$ solana config set -ud    # For devnet
$ solana config set -ul    # For localhost
$ solana config set -ut    # For testnet
```

</Tab>
</Tabs>

## 지갑 생성하기

Nara Solana CLI를 사용하여 트랜잭션을 보내기 전에 NSO이 충전된 Nara Solana 지갑이
필요합니다.

기본 keypair 경로에 keypair를 생성하려면 다음 명령어를 실행하세요:

```terminal
$ solana-keygen new
```

명령이 성공하면 다음과 유사한 출력이 표시됩니다:

```txt title="Example output"
Generating a new keypair

For added security, enter a BIP39 passphrase

NOTE! This passphrase improves security of the recovery seed phrase NOT the
keypair file itself, which is stored as insecure plain text

BIP39 Passphrase (empty for none):

Wrote new keypair to /Users/test/.config/solana/id.json
===========================================================================
pubkey: 8dBTPrjnkXyuQK3KDt9wrZBfizEZijmmUQXVHpFbVwGT
===========================================================================
Save this seed phrase and your BIP39 passphrase to recover your new keypair:
cream bleak tortoise ocean nasty game gift forget fancy salon mimic amazing
===========================================================================
```

<Callout type="info">
  이 명령은 `--force` 플래그를 사용하지 않는 한 기본 위치에 있는 기존 계정을
  덮어쓰지 않습니다.
</Callout>

지갑 주소(공개 키)를 보려면 다음을 실행하세요:

```terminal
$ solana address
```

## NSO 에어드롭

트랜잭션 및 프로그램 배포 비용을 지불하기 위해 지갑에 NSO 에어드롭을 요청하세요.

1. 클러스터를 Devnet으로 설정하세요:

```terminal
$ solana config set -ud
```

2. Devnet NSO 에어드롭 요청하기:

```terminal
$ solana airdrop 2
```

<Callout>

Devnet 에어드롭은 요청당 5 NSO로 제한됩니다. 속도 제한에 도달하거나 오류가
발생하면 대신 [웹 수도꼭지(Web Faucet)](https://faucet.solana.com)를 사용해
보세요.

</Callout>

지갑의 NSO 잔액을 확인하려면 다음 명령어를 실행하세요:

```terminal
$ solana balance
```
