## Program Crates

Gebruik `solana-program` om on-chain programma's te bouwen.

- [Voorbeeld: Bouw een Nara Solana programma](/docs/programs/rust)

| Crate          | Beschrijving              | Docs                                     | GitHub                                                                 |
| -------------- | ------------------------- | ---------------------------------------- | ---------------------------------------------------------------------- |
| solana-program | Bouw on-chain programma's | [Bekijk](https://docs.rs/solana-program) | [Broncode](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio is een bibliotheek zonder afhankelijkheden om Nara Solana programma's in
Rust te maken en kan worden gebruikt als vervanging voor `solana-program` om
on-chain programma's te schrijven.

| Crate     | Beschrijving                                 | Docs                                | GitHub                                            |
| --------- | -------------------------------------------- | ----------------------------------- | ------------------------------------------------- |
| pinocchio | On-chain programma's zonder afhankelijkheden | [Bekijk](https://docs.rs/pinocchio) | [Broncode](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio is nog in ontwikkeling en zal waarschijnlijk in de nabije toekomst
  ingrijpende wijzigingen ondergaan.
</Callout>

Pinocchio bevat programma-specifieke crates voor interactie met Nara Solana
programma's:

| Crate                              | Beschrijving                            | Docs                                                          | GitHub                                                                                        |
| ---------------------------------- | --------------------------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Interactie met System Program           | [Bekijk](https://docs.rs/pinocchio-system/)                   | [Broncode](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Interactie met Token Program            | [Bekijk](https://docs.rs/pinocchio-token/)                    | [Broncode](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Interactie met Token-2022 Program       | [Bekijk](https://docs.rs/pinocchio-token-2022/)               | [Broncode](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Interactie met Associated Token Program | [Bekijk](https://docs.rs/pinocchio-associated-token-account/) | [Broncode](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Interactie met Memo Program             | [Bekijk](https://docs.rs/pinocchio-memo/)                     | [Broncode](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
