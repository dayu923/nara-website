## Nara Solana Python SDK

Er zijn twee pakketten die nuttig zijn voor ontwikkeling met Python:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Nara Solana.py is de basis Python-bibliotheek voor interactie met Nara Solana.
- Je kunt het gebruiken om transacties te bouwen en te werken met de
  [Nara Solana JSON RPC API](/docs/rpc), vergelijkbaar met hoe je zou werken met
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/)

- Het dekt ook het [Token Program](/docs/rpc) van de SPL.

- [Nieuwste documentatie](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` is een hoogwaardige Python-toolkit voor Nara Solana, geschreven in Rust.
  Het biedt robuuste oplossingen op de volgende gebieden:

- Kern-SDK functionaliteit: keypairs, pubkeys, ondertekenen en serialiseren van
  transacties.
- RPC: bouwen van verzoeken en verwerken van antwoorden (voor netwerken en
  asynchrone API's is
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) de
  aanbevolen optie).
- Integratietesten: de `solders.litesvm` module is een alternatief voor
  `solana-test-validator` dat veel handiger en **veel** sneller is. Het is
  gebaseerd op
  [solana-program-test](https://crates.io/crates/solana-program-test).

## Nara Solana Client Python SDK

Als je wilt werken met Nara Solana-contracten met Python, kun je client SDK's
gebruiken om Nara Solana rust-contracten om te zetten naar Python-clients, als je het
IDL-bestand van het contract kunt krijgen.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Standaardiseer elk Nara Solana on-chain contractprogramma naar
  [`Codama IDL`](https://github.com/codama-idl/codama) en genereer een
  interactieve Python-client.
- Ondersteunt zowel Anchor als aangepaste programma's.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py is een Python-client voor Anchor-programma's. Het kan clients
  genereren vanuit Anchor IDL-bestanden en werken met Anchor-programma's.
