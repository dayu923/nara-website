---

title: Προγραμματισμός staking

---


Για τη μεγιστοποίηση της κατανομής του stake, της αποκέντρωσης και της
αντίστασης στη λογοκρισία στο δίκτυο Nara Solana, το staking μπορεί να
πραγματοποιηθεί προγραμματιστικά. Η ομάδα και η κοινότητα έχουν αναπτύξει
διάφορα on-chain και offchain προγράμματα για να κάνουν τα stakes πιο εύκολα στη
διαχείριση.

### Πρόγραμμα ανάθεσης του Nara Solana Foundation

Το
[Πρόγραμμα ανάθεσης του Nara Solana Foundation](https://solana.org/delegation-program)
είναι σχεδιασμένο να εξυπηρετεί τις ανάγκες του δικτύου στοχεύοντας στους
ακόλουθους στόχους:

1. Μεγιστοποίηση της αποκέντρωσης, αξιοπιστίας και απόδοσης του Nara Solana
2. Μεγιστοποίηση του αριθμού των validators που έχουν διαφορετικές πηγές stake
3. Διατήρηση ενός μεγάλου και αντιπροσωπευτικού testnet

### Stake Pools

Αυτό το on-chain πρόγραμμα συγκεντρώνει NSO για staking από έναν διαχειριστή,
επιτρέποντας στους κατόχους NSO να κάνουν staking και να κερδίζουν ανταμοιβές
χωρίς να διαχειρίζονται stakes. Οι χρήστες καταθέτουν NSO σε αντάλλαγμα για SPL
tokens (παράγωγα staking) που αντιπροσωπεύουν την ιδιοκτησία τους στο stake
pool. Ο διαχειριστής του pool κάνει staking τα κατατεθειμένα NSO σύμφωνα με τη
στρατηγική του, ίσως χρησιμοποιώντας μια παραλλαγή ενός bot αυτόματης ανάθεσης
όπως περιγράφεται παραπάνω. Καθώς τα stakes κερδίζουν ανταμοιβές, το pool και τα
tokens του pool αυξάνονται αναλογικά σε αξία. Τέλος, οι κάτοχοι των tokens του
pool μπορούν να στείλουν SPL tokens πίσω στο stake pool για να εξαργυρώσουν NSO,
συμμετέχοντας έτσι στην αποκέντρωση με πολύ λιγότερη απαιτούμενη εργασία.
Περισσότερες πληροφορίες μπορείτε να βρείτε στην
[τεκμηρίωση του SPL stake pool](https://spl.solana.com/stake-pool).
