Αυτή η ενότητα παρέχει κάποιες συνηθισμένες εντολές και παραδείγματα για να σας
βοηθήσει να ξεκινήσετε να χρησιμοποιείτε το Nara Solana CLI.

## Ρύθμιση παραμέτρων Nara Solana

Η ρύθμιση παραμέτρων του Nara Solana καθορίζει τις ακόλουθες μεταβλητές:

- **Αρχείο ρυθμίσεων**: Η διαδρομή προς το αρχείο ρυθμίσεών σας
- **RPC URL & Websocket URL**: Το cluster του Nara Solana στο οποίο το CLI κάνει
  αιτήματα
- **Διαδρομή keypair**: Η διαδρομή προς το προεπιλεγμένο πορτοφόλι Nara Solana
  (keypair) που χρησιμοποιείται για την πληρωμή χρεώσεων συναλλαγών και την
  ανάπτυξη προγραμμάτων. Από προεπιλογή, αυτό το αρχείο αποθηκεύεται στο
  `~/.config/solana/id.json`.

Για να δείτε τις τρέχουσες ρυθμίσεις παραμέτρων σας, εισάγετε την ακόλουθη
εντολή στο τερματικό σας.

```terminal
$ solana config get
```

Μια επιτυχημένη εντολή θα επιστρέψει έξοδο παρόμοια με την ακόλουθη:

```txt title="Example output"
Config File: /Users/test/.config/solana/cli/config.yml
RPC URL: https://api.mainnet-beta.solana.com
WebSocket URL: wss://api.mainnet-beta.solana.com/ (computed)
Keypair Path: /Users/test/.config/solana/id.json
Commitment: confirmed
```

Μπορείτε να αλλάξετε το cluster του Nara Solana CLI με τις ακόλουθες εντολές:

<Tabs items={["Πλήρεις εντολές", "Σύντομες εντολές"]}>

<Tab value="Πλήρεις εντολές">

```terminal
$ solana config set --url mainnet-beta
$ solana config set --url devnet
$ solana config set --url localhost
$ solana config set --url testnet
```

</Tab>
<Tab value="Σύντομες εντολές">

```terminal
$ solana config set -um    # For mainnet-beta
$ solana config set -ud    # For devnet
$ solana config set -ul    # For localhost
$ solana config set -ut    # For testnet
```

</Tab>
</Tabs>

## Δημιουργία πορτοφολιού

Πριν μπορέσετε να στείλετε συναλλαγές χρησιμοποιώντας το Nara Solana CLI, χρειάζεστε
ένα πορτοφόλι Nara Solana με NSO.

Για να δημιουργήσετε ένα keypair στην προεπιλεγμένη διαδρομή keypair, εκτελέστε
την ακόλουθη εντολή:

```terminal
$ solana-keygen new
```

Μια επιτυχημένη εντολή θα επιστρέψει έξοδο παρόμοια με την ακόλουθη:

```txt title="Example output"
Generating a new keypair

For added security, enter a BIP39 passphrase

NOTE! This passphrase improves security of the recovery seed phrase NOT the
keypair file itself, which is stored as insecure plain text

BIP39 Passphrase (empty for none):

Wrote new keypair to /Users/test/.config/solana/id.json
===========================================================================
pubkey: 8dBTPrjnkXyuQK3KDt9wrZBfizEZijmmUQXVHpFbVwGT
===========================================================================
Save this seed phrase and your BIP39 passphrase to recover your new keypair:
cream bleak tortoise ocean nasty game gift forget fancy salon mimic amazing
===========================================================================
```

<Callout type="info">
  Αυτή η εντολή δεν θα αντικαταστήσει έναν υπάρχοντα λογαριασμό στην
  προεπιλεγμένη τοποθεσία, εκτός αν χρησιμοποιήσετε τη σημαία `--force`.
</Callout>

Για να δείτε τη διεύθυνση του πορτοφολιού σας (δημόσιο κλειδί), εκτελέστε:

```terminal
$ solana address
```

## Airdrop NSO

Ζητήστε ένα airdrop NSO στο πορτοφόλι σας για να πληρώσετε για συναλλαγές και
αναπτύξεις προγραμμάτων.

1. Ορίστε το cluster σας στο Devnet:

```terminal
$ solana config set -ud
```

2. Ζητήστε ένα airdrop NSO στο Devnet:

```terminal
$ solana airdrop 2
```

<Callout>

Τα airdrops του Devnet περιορίζουν τα αιτήματα σε 5 NSO ανά αίτημα. Αν
συναντήσετε περιορισμούς ρυθμού ή σφάλματα, δοκιμάστε να χρησιμοποιήσετε το
[Web Faucet](https://faucet.solana.com) αντί αυτού.

</Callout>

Για να ελέγξετε το υπόλοιπο NSO του πορτοφολιού σας, εκτελέστε την ακόλουθη
εντολή:

```terminal
$ solana balance
```
