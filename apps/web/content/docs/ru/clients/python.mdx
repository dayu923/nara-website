Существует [Python SDK](https://github.com/michaelhly/solana-py), который
поддерживает большинство функций для взаимодействия с Nara Solana. Если вы
используете Anchor для своих программ на Nara Solana, вы сможете использовать
[anchor-py](https://github.com/kevinheavey/anchorpy) вместе с вашим idl для
создания клиента, совместимого с Python.

## Установка

```bash title="Terminal"
pip install solders
pip install solana
```

## Дополнительная документация

Пожалуйста, посетите
[документацию Python SDK](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)
для получения дополнительной информации о работе с этим Python SDK для Nara Solana.

- [Последняя документация](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` — это высокопроизводительный Python-инструментарий для Nara Solana,
  написанный на Rust. Он предоставляет надежные решения в следующих областях:

- Основной функционал SDK: keypair, pubkey, подпись и сериализация транзакций.
- RPC: создание запросов и разбор ответов (для сетевых и асинхронных API
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) является
  рекомендуемым вариантом).
- Интеграционное тестирование: модуль `solders.litesvm` является альтернативой
  `solana-test-validator`, которая намного удобнее и **значительно** быстрее. Он
  основан на
  [solana-program-test](https://crates.io/crates/solana-program-test).

## Python SDK клиента Nara Solana

Если вы хотите взаимодействовать с контрактами Nara Solana с помощью Python, вы
можете использовать клиентский SDK для преобразования контрактов Nara Solana на Rust
в Python-клиенты, если у вас есть IDL-файл контракта.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Стандартизируйте любую программу контракта Nara Solana на блокчейне в
  [`Codama IDL`](https://github.com/codama-idl/codama) и создайте интерактивный
  Python-клиент.
- Поддерживает как Anchor, так и пользовательские программы.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py — это Python-клиент для программ Anchor. Он может генерировать
  клиентов из IDL-файлов Anchor и взаимодействовать с программами Anchor.
