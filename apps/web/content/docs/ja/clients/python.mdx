## Nara Solana Python SDK

Pythonで開発する際に役立つパッケージが2つあります：

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Nara Solana.pyはNara Solanaと対話するための基本的なPythonライブラリです。
- トランザクションの構築や[Nara Solana JSON RPC API](/docs/rpc)との対話に使用できます。これは[solana-web3.js](https://github.com/solana-labs/solana-web3.js/)と同様の機能を提供します。

- また、[SPLトークンプログラム](https://spl.solana.com/token)もカバーしています。

- [最新のドキュメント](https://michaelhly.github.io/solana-py/)。

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders`はRustで書かれた、Nara Solana向けの高性能Pythonツールキットです。以下の分野で堅牢なソリューションを提供しています：

- コアSDK機能：keypair、pubkey、トランザクションの署名とシリアライズ。
- RPC：リクエストの構築とレスポンスの解析（ネットワーキングと非同期APIについては、[solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/)が推奨オプションです）。
- 統合テスト：`solders.litesvm`モジュールは
  `solana-test-validator`の代替として、より便利で**はるかに**高速です。これは[solana-program-test](https://crates.io/crates/solana-program-test)に基づいています。

## Nara Solana Client Python SDK

PythonでNara Solanaコントラクトと対話したい場合、コントラクトのIDLファイルを入手できれば、クライアントSDKを使用してNara Solana
RustコントラクトをPythonクライアントにレンダリングできます。

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- あらゆるNara Solanaオンチェーンコントラクトプログラムを
  [`Codama IDL`](https://github.com/codama-idl/codama)に標準化し、インタラクティブなPythonクライアントを生成します。
- Anchorとカスタムプログラムの両方をサポートしています。

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-pyはAnchorプログラム用のPythonクライアントです。Anchor
  IDLファイルからクライアントを生成し、Anchorプログラムと対話することができます。
