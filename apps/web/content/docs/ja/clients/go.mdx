Nara Solanaとのやり取りに必要なほとんどの機能をサポートする[Go SDK](https://github.com/gagliardetto/solana-go)があります。Nara SolanaプログラムにAnchorを使用している場合は、idlと共に[anchor-go](https://github.com/gagliardetto/anchor-go)を使用して、Golang互換のクライアントを生成することができます。

## インストール

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## 詳細なドキュメント

Nara SolanaでこのGo
SDKを使用する方法の詳細については、[Go SDKのドキュメント](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)をご覧ください。
